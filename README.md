# ZapZip

So I was bored.

Instead of doing something productive, I decided to make a platformer game in a single HTML file and somehow it got completely out of hand.

What started as "let's make a character jump" turned into moving platforms, enemies, particles, dashes, checkpoints, sound effects, wall jumps, hidden areas, and a glowing portal at the end of the map.

Welcome to ZapZip.

## What's the Game?

ZapZip is a fast-paced browser platformer where you run, jump, dash, collect crystals, avoid becoming one with the spikes, and somehow survive long enough to reach the portal.

There isn't some deep lore about saving the universe.

You just zip through things.

Hence the name.

## Controls

### Keyboard

Move Left: A / Left Arrow

Move Right: D / Right Arrow

Jump: Space / W / Up Arrow

Dash: Shift

Pause: Esc

### Mobile

Use the on-screen controls and pray.

## Objective

1. Don't fall.
2. Don't touch spikes.
3. Don't run directly into enemies.
4. Collect shiny crystals because shiny things are important.
5. Reach the portal.

That's it.

## Scoring

You get points for:

* Collecting crystals
* Defeating enemies
* Reaching checkpoints

You lose dignity by dying repeatedly in the same place.

## Technical Stuff

The entire game runs in a single HTML file.

No frameworks.

No libraries.

No game engine.

Just HTML, CSS, JavaScript, questionable decisions, and caffeine.

Built using:

* HTML5 Canvas
* Vanilla JavaScript
* Web Audio API
* Pure stubbornness
